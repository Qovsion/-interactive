# 后台接口

## 登录相关

### 登录接口

- 说明

    - 根据邮箱或手机号登录

- 请求方式

    - POST

- 接口地址

    - `/admin/checkLogin`

- 请求参数

    参数编码格式必须为 x-www-form-urlencoded

    | 参数  | 类型   | 说明               |
    | ----- | ------ | ------------------ |
    | email | string | 管理员邮箱或手机号 |
    | pwd   | string | 密码               |

    

- 返回数据示例

    - 成功

        ```json
        { code: 200, message: "登录成功"}
        ```

    - 失败

        ```json
        { code: 201, message: '登录失败' }
        ```
